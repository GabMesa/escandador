Escandidor básico para poetas

Escribe un poema a la izquierda y obtén separación silábica, sílaba tónica y conteo métrico ajustado a la derecha.

Configuración Acento versal, hemistiquio, sinalefa y rima i Soneto: 14 versos, acento versal sugerido 6-10, esquema habitual ABBA ABBA CDC DCD.
Espinela: 10 versos, octosílabo, acento versal sugerido 7, esquema ABBAACCDDC.
Sonetillo: 14 versos octosílabos; puedes usar el mismo esquema del soneto.
Sextina: usa modo de rima Sextina para comparar palabras finales completas.

Hemistiquio

Si este campo está vacío, no se aplica hemistiquio global.

También puedes usar / dentro del verso para separar hemistiquios manualmente.

Sinalefa

Haz clic en cada unión para forzar o romper la sinalefa. El acento versal siempre la corta.

Rima

Si está activo, los grupos separados por espacios se repiten por estrofa. Ej: ABAB ABBA → ABAB ABBA ABAB ABBA...

Haz clic en el color de cada rima para personalizarlo; los colores se guardan para ese poema.
